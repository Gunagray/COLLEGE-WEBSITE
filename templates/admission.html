COURSES OFFERED

BCA


Bachelor of Computer Application (BCA) is an undergraduate program that provides students with a foundation in computer science and programming. The program equips students with the skills and knowledge necessary to pursue careers in software development, database management, web design, and other computer-related fields.

BSC Psychology


BSc Psychology is an undergraduate program that explores the scientific study of the human mind and behavior. Students learn about various theories, research methods, and applications in the field of psychology, preparing them for careers in counseling, education, social work, research, and other related areas.

B.Com
Computer Application


BCom Computer Application is an undergraduate program that combines the study of commerce and computer applications. The program equips students with skills in accounting, taxation, and other commerce-related subjects, along with programming languages, database management, and software development. Graduates can pursue careers in finance, accounting, e-commerce, and other technology-related fields.

BSC.Microbiology


BSc Microbiology is an undergraduate program that focuses on the study of microorganisms, including bacteria, viruses, fungi, and protozoa. The program provides students with a strong foundation in microbiology, preparing them for careers in research, medicine, pharmaceuticals, and other related fields.

BSC
Computer Science


BSc Computer Science is an undergraduate program that covers the fundamentals of computer science, including programming, algorithms, data structures, software development, and computer hardware. Graduates of the program can pursue careers in software development, database management, cybersecurity, and other technology-related fields.

M.Sc
Microbiology


MSc Microbiology is a postgraduate program that focuses on advanced topics in microbiology, including immunology, microbial genetics, biochemistry, and biotechnology. Graduates of the program can pursue careers in research, academia, pharmaceuticals, and other related fields, or pursue further studies in microbiology or related fields.
